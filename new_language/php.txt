在php中
	每行代码的最后要加上分号";"结束。
	PHP 没有声明变量的命令。
	PHP 会根据变量的值，自动把变量转换为正确的数据类型。---所以在使用变量的时候,没有必要强调变量的类型。


一、学习笔记
	1、变量的定义使用“$”符号   $a=5;
		PHP 变量规则：
			变量以 $ 符号开始，后面跟着变量的名称
			变量名必须以字母或者下划线字符开始
			变量名只能包含字母数字字符以及下划线（A-z、0-9 和 _ ）
			变量名不能包含空格
			变量名是区分大小写的（$y 和 $Y 是两个不同的变量）
	2、变量作用域
		local/global/satic/parameter
		函数内定义的变量都是局部变量
		函数之外的变量是全局变量
		
			如果想在函数内使用全局变量，需要在函数内使用global 声明一下变量
			如果想在函数调用之后,仍想使用函数内的局部变量,需要在函数内使用static .比如 static $x =0

		PHP 将所有全局变量存储在一个名为 $GLOBALS[index] 的数组中。 index 保存变量的名称。
			$x=5
			echo $GLOBALS[x]  // 5
	3、echo 和 print语句的区别
		3.1、echo 可以输出一个或多个字符串；
		3.2、print 只允许输出一个字符串，返回值总为 1；
		3.3、echo 输出的速度比 print 快；
		3.4、echo 没有返回值，print 有返回值 1；
	4、PHP EOF(heredoc) 使用说明
		具体可以看看菜鸟教程或者直接百度。感觉不写php，代码应该用不到
	5、php中的数据类型
		String（字符串）, 
		Integer（整型）,
		Float（浮点型）, 
		Boolean（布尔型）,
		Array（数组）, 
		Object（对象）,
		NULL（空值）
		
		php中   浮点数之间可以进行加减运算       $x=true  $y=false echo $x+$y  // 1
				浮点数和整数之间也可以进行运算   $x=true  $y=2 echo $x+$y  // 3
				true:1
				false:0
	6、定义常量(使用define()函数)----常量是全局的
		bool define(string $name,mixed $value ,true ) // true表示使用时,不区分大小写
		例如:
			define("GREETING","欢迎到此一游");
			echo GREETING;
	7、关联数组是使用自己分配给数组的制定的键的数组。比如:
			$age=array("Peter"=>"35","Ben"=>"37","Joe"=>"43");
		or:
			$age['Peter']="35";
			$age['Ben']="37";
			$age['Joe']="43";
	8、对php数组进行排序的时候
		看到的有两种
		
		数值数组 
				  升序排序
				  降序排列
		关联数组
			根据关联数组的键进行升序和降序排序
			根据关联数组的值进行升序和降序排序
	9、定义函数 
		function funcName(param1,param2){
			function body
		}

		函数的返回值没有不用声明,如果函数需要返回,在function body中添加一个return就好

		例如: function writeName($name,$age){
			return "my name is".%name." I am ".$age
		}
	10、php中的魔鬼变量(它们的值随着它们在代码中的位置改变而改变)
		_LINE_:文件中的当前行号
		__FILE__:文件的完整路径和文件名。如果用在被包含文件中，则返回被包含的文件名。
		__DIR__:文件所在的目录。它等价于 dirname(__FILE__)。除非是根目录，否则目录中名不包括末尾的斜杠。
		__FUNCTION__:函数名称（函数名的大小写和版本有关系）
		__CLASS__:类的名称
		__TRAIT__:(这个有点复杂,暂时没看懂)
		__METHOD__:类的方法名,返回该方法定义时的方法名(区分大小写)
		__NAMESPACE__:当前命名空间的名称（区分大小写）

二、和其他语言的对比
2.1、字符串链接
	php中使用“,”或者"."连接。
	golang中使用"+"连接
2.2、php中的数据类型
	php中没有map/Slice/channel/struct
		只用String/Integer/Float/Boolean/Array/Object/Null
2.3、对于循环
	golang中没有while，do..while循环和foreach循环(foreach就是遍历数组的时候)
	php中有while和for循环
	
	在循环的时候,大括号的位置和小括号也得加进去
